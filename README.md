# Nest Test Project

- 단위 테스트, 종단 테스트 구현
- Guard 구현
- Jwt 설정
- Database 설정
- Config 전역 설정
- ApiResponse Form 생성
- dto 상속 및 typeorm NM, 1N 설정 시 활용 가능 방법 적용

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
